# iStore MERN E-commerce App

A full-stack e-commerce platform with product browsing, search, cart, authentication, payments, and admin capabilities.

## Features
- Product listing and search
- Authentication and authorization
- Cart and checkout
- Stripe payment integration
- Admin dashboard
- Image uploads
- Order management

## Tech Stack
- Frontend: React, Redux
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Payments: Stripe
- Media: Cloudinary
- Auth: JWT

## Architecture
- `frontend/` contains React client
- `server/` contains REST APIs and business logic
